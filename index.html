About Me


My name is Adrian Munoz, I was born and raised is Los Angeles CA. I currently live with my little girl in the beautiful high Desert known as Palmdale CA. Growing up I always had a natural talent for drawing and illustrating, at one point I was even privileged enough to pitch a cartoon to Cartoon Network Studios (it didn't get picked up).

Since then I received my degree in Game Software Development and published my first game in the Play store titled Death Bird. I also purchased the Oculus Rift Developers Kit 2 in order to develop my first Virtual reality Game using the Unity 3d engine, in my spare time. For the past two years I've been learning and working with web development. I started off learning the backend first with Ruby on Rails and MVC framework, then learned HTML, CSS and JavaScript and currently in the process of earning a Certificate in Angular JS.

I spend my free time with my 11 year old daughter, I just began teaching her the basics of HTML and CSS so we could build a website together. At a young age my father taught me what it means to work hard in the construction field pouring concrete. I am passing down that same work ethic along with a skill that will help her as an adult like my father did for me.



Experiences


I began working as an artist and programmer for A Manaical Game. I used Photoshop and illustrator to create content. Sprite sheets were created with flash to test the framerates for animation. I also would animate story boards to show how the game or cut scenes might look before we began the serious work.

I then found a job as a QA analyst trainee where I was first introduced to HTML, CSS and SQL. My job was to look through wire frame documents and test out the functionality of the web page. We were in charge of creating task cases, running them then pass or fail. If a test case failed we'd write a report on possible causes. Setting Automated test cases was a common practice. They would run during the night, in the mornings we'd check to see which passed or failed. If any failed we'd re run the tests manually. I worked in an agile development environment, therefore we had constant communication with the Development team.

I realize that want to develop more than test so I began working on a game with the goal of publishing it in the Play Store I kept it simple with the only goal of beating your high score I took the idea of Flappy Bird but with missiles and my own original artwork. I use Photoshop and illustrator once more to make it appealing to the eye C sharp was used for the programming language and unity 3D for the framework to render and package the APK file for the Play Store

I then began working as a software developer for kraferd Incorporated in Fullerton California they put me through an online boot camp to learn Ruby on Rails. I was once again reintroduced to sql by using mySQL with rails. here I learned about the MVC framework which separated the views of the page from the actions and database. I work with a restful environment we used to crud to create read update and delete the information from the database with the HTTP protocol verbs. here I work with HTML CSS and bootstrap for the front end and a bit of json and Ajax for partial view updating. We used Github and heroku for version control, and operated on a Linux Desktop Ubuntu Distro.
